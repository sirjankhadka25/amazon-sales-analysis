# Amazon Sales Analysis Dashboard

## Overview
Analysis of 100,000 Amazon orders across 5 countries generating $91.83M in revenue.

## Key Insights
- Electronics is the top revenue category
- United States is the largest market
- Cancellation rate is healthy at 3%
- Higher discounts do NOT drive more sales
- Peak revenue month was May 2024

## Tools Used
- Python (pandas, numpy, matplotlib, seaborn)
- Power BI
- Jupyter Notebook

## Files
- `sales_analysis.ipynb` — Python analysis
- `Amazon.csv` — Raw dataset
- `dashboard.pdf` — Power BI dashboard
